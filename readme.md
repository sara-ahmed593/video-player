## Description

This is a responsive video player web application built with HTML, SCSS, and JavaScript. It allows users to play, pause, and navigate between videos in a playlist. Users can control the video progress, adjust the volume, mute/unmute the sound, and switch to full-screen mode.       

## Features


- The web application  have a video player that can play, pause, and stop video playback.
- Users can navigate through the video by clicking the progress bar.
- The web application  display the current time and total time of the video.
- The application  include volume controls that allow users to adjust the volume of the video.
- The application  include a control to allow users to toggle between full-screen and normal mode.
- Navigate to the next or previous video in their playlist
